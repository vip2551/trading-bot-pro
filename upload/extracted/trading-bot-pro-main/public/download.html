📋 خطوات التشغيل:

1. أنشئ مجلد جديد: mkdir trading-bot && cd trading-bot
2. أنشئ ملف package.json والصق المحتوى
3. شغل: bun install
4. أنشئ الملفات الأخرى (src/, prisma/, public/)
5. أنشئ .env من .env.example
6. شغل: bun run db:push && bun run prisma generate
7. شغل: bun run dev
8. افتح: http://localhost:3000
