📋 طريقة التثبيت:

  1. نزّل ملف ZIP من الرابط أعلاه
  2. اذهب لمجلد المشروع: C:\Users\MOHAMMED\trading-bot-pro
  3. استخرج محتويات ملف ZIP في المجلد (استبدال الملفات الموجودة)
  4. افتح موجه الأوامر (Command Prompt) في المجلد
  5. نفذ الأوامر التالية: 
    git add .
git commit -m "إصلاح الأخطاء"
git push origin main
  6. Railway سيقوم بنشر التحديثات تلقائياً!

⚠️ ملاحظة مهمة:

بعد الرفع، تأكد من إضافة متغيرات البيئة في Railway:

  • DATABASE_URL - من PostgreSQL
  • NEXTAUTH_SECRET - المفتاح الذي أنشأته
  • NEXTAUTH_URL - رابط موقعك
  • NODE_ENV = production
